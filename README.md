## 과제 소개 🌟
- 원티드의 실제 페이지와 동일하게 만드는 과제
<br>

- 상단 GNB(Global Navigation Bar) 구현
  - 마크업 작업(HTML, CSS)만 작업
  - 메뉴클릭, hover 등의 이벤트는 구현 생략
<br>

- 슬라이드(캐러샐) 영역 구현
  - 라이브러리를 사용하지 않고 로직 구현
<br>

- 웹페이지 넓이에 따른 3단계의 반응형 레이아웃 구현

<br>
